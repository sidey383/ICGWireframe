Задача 4 – ICGWireframe

Приложение для отображения в виде проволочной модели тела вращения с образующей, строящейся по задаваемым опорным точкам с помощью B-сплайна.

Взаимодействия с основным окном

  1. ЛКМ - вращение фигуры по осям XY
  2. ПКМ - вращение по оси Z
  3. Колесико мыши - масштабировани

Взаимодействия с редактором

  1. ЛКМ по свободному месту - двигать камеру
  2. ЛКМ по точке - перемещение точки
  3. shift + ЛКМ по точке - двигать всю фигуру
  4. ПКМ - поставить точку
  5. shift + ПКМ по точке - удалить точку
  6. Колесико мыши - масштабировани
